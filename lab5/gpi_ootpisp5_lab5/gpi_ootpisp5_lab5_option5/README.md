## Вывод заработной платы за месяц по каждому работнику

```sql
SELECT gpi_month, gpi_family, gpi_name, gpi_otch, gpi_summa
FROM gpi_fio INNER JOIN gpi_vid
ON gpi_fio.gpi_id = gpi_vid.gpi_id
ORDER BY gpi_month DESC;
```

## Вывод вида работ работника за месяц

```sql
SELECT gpi_month, gpi_family, gpi_name, gpi_otch, gpi_vid
FROM gpi_fio INNER JOIN gpi_vid
ON gpi_fio.gpi_id = gpi_vid.gpi_id
ORDER BY gpi_month DESC;
```
